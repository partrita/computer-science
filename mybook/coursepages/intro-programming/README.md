## CS50P: 파이썬을 활용한 프로그래밍 입문

이 과정은 하버드 대학교(Harvard University)의 CS50 팀에서 개발했습니다.

> 파이썬(Python)이라는 언어를 사용한 프로그래밍 입문 과정입니다. 코드를 읽고 쓰는 방법뿐만 아니라 테스트하고 "디버깅(debug)"하는 방법도 배웁니다. 파이썬을 특별히 배우고자 하는, 프로그래밍 경험 유무와 상관없이 모든 학생들을 위해 고안되었습니다. 함수, 인수, 반환값부터 시작해 변수, 자료형, 조건문과 불리언 식, 반복문에 대해 알아봅니다. 나아가 예외 처리 방법, 버그를 찾고 고치는 방법, 단위 테스트(unit test) 작성 방법을 배우며; 서드파티 라이브러리를 활용하고 정규 표현식을 이용해 데이터를 검증 및 추출하는 법; 클래스, 객체, 메서드, 속성(property)을 통해 현실 세계의 개체들을 모델링하는 방법, 그리고 파일을 읽고 쓰는 법까지 다룹니다. 풍부한 실습 기회가 주어지며, 연습 문제들은 현실 세계의 프로그래밍 문제들에서 영감을 받았습니다. 웹 브라우저 외에 별도의 소프트웨어가 필요하지 않지만, 각자의 개인 PC나 Mac에서 코드를 짤 수도 있습니다.

**링크**: <https://cs50.harvard.edu/python/>

**참고**: 이 강좌는 CS50 또는 CS50x 과정과는 다릅니다. 원조 CS50은 OSSU 커리큘럼에 속해 있지 않습니다. 다만 여러분이 예전에 이미 CS50을 수료한 적이 있다면 이 과정을 무시하고 바로 다음 커리큘럼으로 넘어가셔도 좋습니다.

### 문제 해결 패치안 (Fixes)

1. BeautifulSoup4 과제를 진행하고 있을 무렵 만약 파이썬 버전을 3.10+ 에 놓고 작업 중이라면 Collections 라이브러리 참조 구문이 거부되는 골치 아픈 에러 현상이 터질 것입니다. 이때 당면한 해결책을 전달해 드립니다. 억지로 이해하려 들지 마시고 그냥 프로그램 맨 꼭대기 불러오기 (imports block) 공간에 다음 줄을 몰래 복사해 집어넣으십시오:

```python
import collections
collections.Callable = collections.abc.Callable

from bs4 import BeautifulSoup 
```

이렇게 하면 꼬여있던 콜러블 (Callable) 인터페이스 호환성 문제가 말끔히 치유되어 무사히 프로그래밍이 정상 작동할 것입니다.
