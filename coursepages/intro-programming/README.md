# 프로그래밍 입문

[CS 입문 과정](../intro-cs/README.md)이 따라가기 어렵다면 여기에서 시작하십시오. 이 과정들은 자율 학습 방식으로 진행되므로 필요에 따라 주당 학습 시간을 조절할 수 있습니다.

## CS50P: 파이썬을 이용한 프로그래밍 입문

이 과정은 하버드 대학교의 CS50 팀에서 개발했습니다.
> 파이썬이라는 언어를 사용한 프로그래밍 입문 과정입니다. 코드를 읽고 쓰는 방법뿐만 아니라 테스트하고 "디버깅"하는 방법도 배웁니다. 프로그래밍 경험이 있거나 없는 학생 중 특히 파이썬을 배우고 싶은 학생들을 위해 설계되었습니다. 함수, 인자, 반환 값 (세상에!); 변수와 자료형; 조건문과 부울 표현식; 그리고 반복문에 대해 배웁니다. 예외 처리, 버그 찾기 및 수정, 단위 테스트 작성 방법; 타사 라이브러리 사용 방법; 정규 표현식을 사용한 데이터 유효성 검사 및 추출 방법; 클래스, 객체, 메서드 및 속성을 사용한 실제 세계 엔티티 모델링 방법; 그리고 파일 읽고 쓰는 방법을 배웁니다. 많은 연습을 위한 실습 기회가 제공됩니다. 실제 프로그래밍 문제에서 영감을 받은 연습 문제입니다. 웹 브라우저 외에는 소프트웨어가 필요 없으며, 자신의 PC나 Mac에서 코드를 작성할 수도 있습니다.

**링크**: <https://cs50.harvard.edu/python/>

**참고**: 이 과정은 CS50 또는 CS50x와는 *다릅니다*. CS50은 OSSU 커리큘럼의 일부가 아닙니다. 즉, CS50을 완료했다면 이 과정을 건너뛰고 다음 과정으로 넘어갈 수 있습니다.

### 수정 사항

1. BeautifulSoup4 강의를 진행 중이고 파이썬 3.10 이상 버전에서 Collections 라이브러리를 참조하는 오류가 발생하는 경우 해결 방법이 있습니다. 이해하지 못해도 괜찮습니다. 코드의 import 블록 앞에 다음을 추가하십시오:

```python
import collections
collections.Callable = collections.abc.Callable

from bs4 import BeautifulSoup
```

이렇게 하면 호환성 문제가 해결되고 코드가 실행될 것입니다.
